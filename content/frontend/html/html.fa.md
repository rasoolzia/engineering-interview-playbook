## سوال 1

**آیدی**: html-001  
**عنوان**: تفاوت `<div>` و `<span>` چیست؟  
**سختی**: آسان  
**دسته‌بندی**: اصول اولیه

### جواب

`<div>` و `<span>` هر دو عناصر عمومی (Generic) هستند، اما در نحوه نمایش و کاربرد تفاوت دارند.

**`<div>`**

- عنصر block-level
- کل عرض موجود را اشغال می‌کند
- از خط جدید شروع می‌شود
- برای گروه‌بندی بخش‌های بزرگ‌تر استفاده می‌شود

**`<span>`**

- عنصر inline
- فقط به اندازه محتوای خود عرض می‌گیرد
- از خط جدید شروع نمی‌شود
- برای بخش‌های کوچک داخل متن استفاده می‌شود

## سوال 2

**آیدی**: html-002  
**عنوان**: هدف از استفاده از `doctype` چیست؟  
**سختی**: آسان  
**دسته‌بندی**: اصول اولیه

### جواب

اعلان `<!DOCTYPE>` به مرورگر اعلام می‌کند که سند با چه نسخه‌ای از HTML نوشته شده است.

اهداف اصلی آن:

- فعال‌سازی حالت استاندارد (Standards Mode)
- جلوگیری از فعال شدن حالت سازگاری قدیمی (Quirks Mode)
- تضمین رفتار صحیح CSS و Layout

## سوال 3

**آیدی**: html-003  
**عنوان**: تفاوت عناصر Block و Inline چیست؟  
**سختی**: آسان  
**دسته‌بندی**: اصول اولیه

### جواب

تفاوت این عناصر در نحوه نمایش آن‌ها در صفحه است.

**عناصر Block**

- از خط جدید شروع می‌شوند
- کل عرض موجود را اشغال می‌کنند
- می‌توانند شامل عناصر Block یا Inline باشند

مثال: `<div>`، `<p>`، `<section>`

**عناصر Inline**

- از خط جدید شروع نمی‌شوند
- فقط به اندازه محتوای خود عرض می‌گیرند
- معمولاً داخل متن استفاده می‌شوند

مثال: `<span>`، `<a>`، `<strong>`

## سوال 4

**آیدی**: html-004  
**عنوان**: عناصر معنایی (Semantic) در HTML چه هستند؟  
**سختی**: آسان  
**دسته‌بندی**: اصول اولیه

### جواب

عناصر Semantic عناصری هستند که معنای مشخصی درباره محتوای خود ارائه می‌دهند.

مزایا:

- بهبود دسترسی‌پذیری
- بهبود سئو
- خوانایی و نگهداری بهتر کد

نمونه‌ها:

- `<header>`
- `<nav>`
- `<main>`
- `<article>`
- `<section>`
- `<footer>`

برخلاف `<div>`، این عناصر نقش مشخصی را توصیف می‌کنند.

## سوال 5

**آیدی**: html-005  
**عنوان**: تفاوت `id` و `class` چیست؟  
**سختی**: آسان  
**دسته‌بندی**: اصول اولیه

### جواب

`id` و `class` هر دو برای شناسایی عناصر استفاده می‌شوند اما تفاوت‌هایی دارند.

**`id`**

- در هر صفحه باید یکتا باشد
- برای هدف‌گیری یک عنصر خاص استفاده می‌شود
- معمولاً در JavaScript یا لینک‌های لنگر (anchor) کاربرد دارد

**`class`**

- می‌تواند روی چندین عنصر استفاده شود
- برای گروه‌بندی عناصر با استایل مشابه استفاده می‌شود
- رایج‌ترین روش هدف‌گیری در CSS است

هر عنصر می‌تواند چندین class داشته باشد اما فقط یک id دارد.
